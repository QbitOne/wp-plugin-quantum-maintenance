Easy maintenance mode plugin for your website.

== Description ==

This plugin helps you managing maintenance work on your website. You can activate the maintenace mode for particular webpages.

== Installation ==

1. Upload `quaintenance` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
